CCNA Basic Office Network

Project Overview

This lab demonstrates a simple office network built using Cisco Packet Tracer.

## Topology

1 Router
1 Switch
3 PCs

## IP Addressing

Router
192.168.1.1 /24

PC1
192.168.1.10

PC2
192.168.1.11

PC3
192.168.1.12

Default Gateway
192.168.1.1

## Verification

* ping between PCs
* show ip interface brief

## Skills Demonstrated

* Basic router configuration
* IP addressing
* Network connectivity testing
